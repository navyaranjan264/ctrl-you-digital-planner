# Ctrl+You – Digital Planner (Java Desktop Application)

Ctrl+You is a full-featured desktop productivity application built using **Java Swing** and **MySQL (JDBC integration)**.

It helps students organize notes, manage tasks, visualize ideas through interactive mind maps, and maintain mental wellness — all in one application.

---

##  Features

###  Notes Management
- Create, edit, and update notes
- Stored persistently using MySQL
- Dynamic loading from database

###  Interactive Mind Map Editor
- Drag & drop nodes
- Double-click to edit
- Save and load mind maps from database

###  Pomodoro Focus Timer
- 25-minute focus session
- Start, pause, reset functionality

###  Task Checklist
- Add tasks
- Mark tasks as completed
- Delete tasks
- Data stored in database

###  Mood Tracker
- Daily mood check-in
- Logs stored in database
- Dynamic dashboard stats

###  Reflection Journal
- Save daily journal entries
- Persistent storage using MySQL

---

##  Tech Stack

- Java (Core + OOP)
- Java Swing (GUI Development)
- MySQL
- JDBC
- Event-driven Programming
- Object-Oriented Architecture
